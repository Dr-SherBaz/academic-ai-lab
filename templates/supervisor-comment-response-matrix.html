Supervisor Comment Response Matrix Template

May 1, 2026

About This Template

This Excel-based response matrix helps you:

  • Log every supervisor comment systematically
  • Plan specific actions for each comment
  • Track completion status
  • Generate a summary report to accompany your revised submission

Template Structure

The matrix includes the following columns:

| Column | Description |

|--------|-------------|

| Comment ID | Auto-generated unique identifier |

| Supervisor Comment | Original verbatim comment |

| Page / Section | Where the comment applies |

| Category | Structural, Methodological, Content, Formatting, Language |

| Priority | Must-Fix, Should-Address, Optional |

| Proposed Action | Your planned response |

| Section(s) Affected | Chapters or sections requiring changes |

| Requires Data | Yes / No |

| Status | Pending, In Progress, Complete |

| Notes | Additional observations or questions for supervisor |

How to Use

  • Download and save a copy for your project
  • Enter every supervisor comment in the second column
  • Categorise and prioritise each comment
  • Write your proposed action for each item
  • Update the status as you work through revisions
  • Print or export the completed matrix to accompany your revised draft

Why This Works

Supervisors appreciate seeing that every comment was systematically addressed.

A completed matrix demonstrates thoroughness and makes the revision process

transparent — which can lead to faster approval and fewer rounds of revision.
